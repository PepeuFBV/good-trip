# Dotfiles

My personal configuration files for various applications and tools.

## Installation

To set up my dotfiles on your system, you can clone the repository and create symbolic links to the configuration files in your home directory.

```bash
git clone [repository_url] ~/dotfiles
cd ~/dotfiles
./install.sh
```

## Included Configurations

-   `.zshrc` - Zsh shell configuration
